How to Run This Application

  1. Windows:
    1. Open Command Prompt (Win + R, type cmd, press Enter)
    2. Navigate to your JAR folder: cd "C:\path\to\folder"
    3. Run: java -jar ShoppingCart.jar
  2. Mac/Linux: Run java -jar ShoppingCart.jar in Terminal
